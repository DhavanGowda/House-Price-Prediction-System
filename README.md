# 🏠 House Price Prediction using Machine Learning

## 📌 Project Overview

The House Price Prediction System is a Machine Learning-based web application developed using Python and Streamlit. The main objective of this project is to predict house prices accurately based on property features such as location, total square feet, number of bathrooms, and BHK.

The system uses the Random Forest Machine Learning algorithm for generating accurate predictions and provides market analysis through graphical visualization. The application also allows users to download a complete prediction report in PDF format along with graphical analysis.
